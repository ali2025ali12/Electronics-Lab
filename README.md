# 🔧 Electronics Lab

A hands-on electronics laboratory containing practical projects using Arduino, ESP32, RF modules, sensors, and embedded systems.

---

## 📌 Overview

This repository is a collection of real-world electronics experiments and DIY projects focused on learning, prototyping, and building practical embedded systems.

It includes:
- Microcontroller projects (Arduino, ESP32)
- Wireless communication (NRF24L01)
- Sensor-based systems
- Basic analog and digital circuits
- Hardware testing tools

---

## 🚀 Featured Projects

### 📡 Wireless Communication
- NRF24L01 transmitter/receiver systems
- Remote control systems (RC style projects)
- Data telemetry experiments

### 🤖 Embedded Systems
- Arduino Nano sensor tests
- ESP32 IoT experiments
- LCD display interfaces (16x2 I2C)

### 🔧 Electronics Tools
- Continuity / wire tester
- Signal generator experiments
- Basic measurement tools

---

## 🧠 Learning Goals

This repository is built to improve skills in:
- Embedded C/C++ programming
- Circuit design and debugging
- Wireless communication systems
- Real-world electronics troubleshooting
- Hardware prototyping

---

## 🛠️ Hardware Used

- Arduino Nano / Uno
- ESP32
- NRF24L01 2.4GHz modules
- LCD 16x2 (I2C)
- Various sensors (DHT11, etc.)
- Breadboard & custom PCBs
